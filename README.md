# 我的个人空间 💜

一个唯美紫色系的个人展示网站，包含教育背景、工作经历、技能兴趣、日志、计划和联系方式等模块。

## 🌟 网站功能

- **首页**：个人简介、导航入口、最新动态
- **关于我**：教育背景时间轴、工作经历、获奖证书画廊
- **技能与兴趣**：技能进度条、兴趣爱好卡片
- **日志**：文章列表、分类筛选、搜索功能
- **计划**：目标看板、进度追踪、统计展示
- **联系我**：联系表单、社交媒体链接

## 🎨 设计特点

- 唯美紫色系配色方案
- 毛玻璃效果设计
- 圆润柔和的UI风格
- 丰富的交互动效
- 完全响应式布局

## 🚀 在线访问

网站部署在 GitHub Pages 上：
https://[你的用户名].github.io/[仓库名]/

## 📝 技术栈

- HTML5
- CSS3 (Tailwind CSS + 自定义样式)
- JavaScript (ES6+)
- Font Awesome 图标
- Google Fonts 字体
