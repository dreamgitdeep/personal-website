# 📝 网站编辑指南

## 如何修改网站内容

### 1. 修改名字

**文件**: `index.html`

**找到这行** (第46行):
```html
<h1 class="hero-title">你好，我是 <span class="highlight">小雅</span></h1>
```

**改成你的名字**:
```html
<h1 class="hero-title">你好，我是 <span class="highlight">你的名字</span></h1>
```

---

### 2. 修改个人简介

**文件**: `index.html`

**找到这行** (第51-53行):
```html
<p class="hero-description">
    热爱生活，喜欢探索新事物。在这里记录我的成长、分享我的故事。
</p>
```

**改成你的简介**:
```html
<p class="hero-description">
    这里写你自己的简介，想写什么就写什么。
</p>
```

---

### 3. 修改打字机文字

**文件**: `js/main.js`

**找到这段代码**:
```javascript
const texts = [
    '热爱生活 ✨',
    '喜欢分享 💜',
    '永远在学习 📚',
    '记录美好时光 🌸'
];
```

**改成你想要的文字**:
```javascript
const texts = [
    '你的文字1',
    '你的文字2',
    '你的文字3',
    '你的文字4'
];
```

---

### 4. 更换头像

**方法A - 使用网络图片**:
找到图片链接，替换第44行:
```html
<img src="你的图片链接" alt="个人头像">
```

**方法B - 使用本地图片**:
1. 把你的照片放到 `images/profile/` 文件夹
2. 改名为 `avatar.jpg`
3. 修改代码:
```html
<img src="images/profile/avatar.jpg" alt="个人头像">
```

---

### 5. 修改联系方式

**文件**: `contact.html`

**找到并修改**:
- 邮箱: 搜索 `xiaoya@example.com` 改成你的邮箱
- 城市: 搜索 `中国 · 杭州` 改成你的城市
- 抖音/B站链接: 搜索 `#` 改成你的真实链接

---

### 6. 添加新日志

**文件**: `blog.html`

**复制一个日志卡片**，修改内容:
```html
<article class="blog-card" data-category="life">
    <div class="blog-image">
        <img src="图片链接" alt="标题">
        <span class="blog-category life">生活</span>
    </div>
    <div class="blog-content">
        <div class="blog-meta">
            <span class="blog-date">
                <i class="far fa-calendar"></i> 2026-02-20
            </span>
        </div>
        <h3 class="blog-title">你的日志标题</h3>
        <p class="blog-excerpt">日志摘要内容...</p>
        <div class="blog-tags">
            <span class="tag">#标签1</span>
        </div>
    </div>
</article>
```

---

## 💡 编辑小技巧

1. **用 VS Code 编辑** - 免费下载，有颜色提示
2. **改完保存** - Ctrl+S 保存文件
3. **刷新看效果** - 在浏览器按 F5 刷新
4. **改错了怎么办** - 重新复制原文件即可

## ❓ 常见问题

**Q: 修改后网页没变化？**
A: 按 Ctrl+F5 强制刷新，或清除浏览器缓存

**Q: 图片显示不出来？**
A: 检查图片路径是否正确，建议用英文命名

**Q: 改错了怎么恢复？**
A: 可以重新从 GitHub 下载原始文件
